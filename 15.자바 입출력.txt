데이터 -> -------	-> 프로그램 변수...
프로그램 ---------- 출력(화면, 파일)

1. 스트림 
- 가상의 통로 ->입력, 출력

1) 입력스트림과 출력 스트림
입력 스트림 InputStream
				- FileInputStream
				- FileReader
				- BufferedInputStream
				- BufferReader
				- InputStreamReader
				
출력 스트림 OutputStream
				- FileOutputStream
				- FileWriter
				- BufferedOutputStream
				- BufferedWriter
				- OutputStreamWriter

2) 바이트 단위 스트림과 문자단위 스트림
	바이트 단위 스트림(1바이트)
		- FileInputStream, FileOutputStream, BufferedInputStream, BufferedOutputStream
	
	문자단위 스트림(2바이트)
		- FileReader, FileWriter, BufferedReader, BufferedWriter

3) 기반 스트림과 보조 스트림
	기반 스트림
		- 직접 읽고 쓰는 스트림
		FileInputStream, FileReader, FileOutputStream, FileWriter
		
	보조 스트림
		- 직접 읽고 쓰는 기능 X -> 기반 스트림에 + 기능을 추가
		InputStream, OutputStreamWriter - 바이트 단위 스트림 -> 문자 단위 스트림으로 변경
		BufferedInputStream,BufferedOutputStream
		BufferedReader, BufferedWriter
		
2. 표준입출력
	콘솔에서 입력, 출력
	System 클래스
		static PrintStream out - 콘솔에서 출력
		static InputStream in - 콘솔에서 입력
		static PrintStream err - 콘솔에서 에러를 출력
		
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		-> 문자 단위로 쉽게 입력 받을 수 있는 클래스 Scaner 클래스
